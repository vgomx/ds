Avatar — a square ink tile with mono initials (or a cropped image). An extension primitive (the source library has no avatar); kept restrained and on-system.

```jsx
<Avatar name="Vitor Gomes" />
<Avatar name="Vitor Gomes" src="/me.jpg" size="lg" />
<Avatar name="VG" shape="circle" size="sm" />
```

Square (2px radius) by default to match the system's sharp geometry; `shape="circle"` when a round avatar is needed. Sizes `sm | md | lg`.
