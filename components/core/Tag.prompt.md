Tag — a mono, uppercase category chip for disciplines, years and labels. Outline by default.

```jsx
<Tag>Visual Design</Tag>
<Tag tone="solid">Featured</Tag>
<Tag tone="subtle">2026</Tag>
<Tag onDark>Case Study</Tag>
```

Tones: `outline` (hairline border — the workhorse), `solid` (ink fill), `subtle` (grey wash). Sizes `sm | md`. Set `onDark` when placing on an ink surface. For live status with a colored dot, use `Badge` instead.
