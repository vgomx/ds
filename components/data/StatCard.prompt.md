StatCard — a big figure over a mono caption. Drop it inside an ink `Card` to give the number the accent.

```jsx
<Card tone="ink"><StatCard value="12+" label="Years shipping product" onDark /></Card>
<StatCard value="40" label="Projects shipped" />
```

`onDark` tints the figure with the on-dark accent and lightens the caption. Pair with `Card` for the boxed treatment, or use bare in a stat row.
